Catalogador de Estratégias IQ Option (2021)

O Catalogador de estratégias foi um programa feito a pedido de uma "sala de sinais", que queriam automatizar o processo, da qual, eles estudavam e mandavam por chat (telegram) a melhor hora para realizar a aposta.

Para isso, aprendi as estratégias mais usadas naquele mercado e para aplica-las corretamente utilizei o serviço de API da IQ Option, que me disponibilizava todas as "velas" em um determinado periodo de tempo.

Ao resgata-las eu aplicava a todas as estratégia e mostrava a porcentagem de acerto de cada uma.

As linguagens utilizadas foram:

  • Python e QT Designer

    Em python era feito o recebimento dos dados da API e a lógica para as estratégias, que após serem feitas, eram atualizadas na tela do QT Designer,

  • PHP

    Recebia e enviada informações, por meio de POST e GET do Python, dos usuários cadastrados.

  • MySQL

    Guardava as informações sobre os usuários, versão da aplicação e a key de registro.
